# DeepClick 产品定位基准（博客叙事用）

## 一句话定位
DeepClick = 面向 Meta/FB 投流场景的**点击后链路优化平台**，通过回流与再曝光机制，把原本流失的点击转成更多安装与转化。

## 核心价值主张（所有内容必须回归这三点）
- **CVR 提升**（转化率提高）
- **CPA 下降**（单次转化成本降低）
- **增量转化净收益 > 工具成本**

## DeepClick 解决的问题
1. 广告有点击，但转化率低
2. 用户首次点击后决策不充分导致流失
3. 已买到的流量缺少二次利用机制
4. click-to-conversion 链路缺乏系统性后优化能力

## 核心能力
- 回流链接 / 落地页
- 受众回流 / 再曝光
- PWA 回流 / 推送回流
- 智能绿盾与风控拦截
- 老客落地页 / 钱包能力

## 目标客户
- **广告投放负责人**：关注投放结果、ROI、CPA 是否下降
- **增长负责人**：关注转化是否有增量
- **业务 Owner**：关注净收益

## 竞品区隔
DeepClick **不是**：纯 tracker、落地页编辑器、传统投放后台
DeepClick **是**：回访态策略 + 再曝光编排 + 转化闭环系统
